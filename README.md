## ⏱️ 日々の残り時間 - カウントダウンタイマー

https://velaciela-ringWing2007.github.io/countdownTimerPage.html

日常の様々なイベントまでの残り時間をリアルタイムで表示します。

## ✨ 機能

- **複数タイマー同時表示** - 起床時刻、勤務開始、定時退勤、就寝時間など複数のカウントダウンを一覧表示
- **4種類のタイマー形式**
  - 毎日（例: `08:20:00`） - 毎日その時刻までカウントダウン
  - 毎週（例: 月曜 09:00） - 毎週指定曜日にリセット
  - 毎月（例: 25日 18:00） - 毎月指定日にリセット
  - 特定日時（例: `2024-01-01T00:00:00`） - 特定の日時までカウントダウン
- **グループ機能** - 複数のタイマーセット（仕事用、プライベート用など）を切り替え可能
- **視覚的なアラート** - 残り時間に応じて色が変化
  - 🔵 青: 1時間以上
  - 🟠 オレンジ: 30分〜1時間
  - 🔴 赤: 15分〜30分
  - 🔴 点滅: 15分未満
- **設定のカスタマイズ** - GUI設定画面またはJSON形式で編集可能
- **エクスポート/インポート** - JSONファイルで設定をバックアップ・復元
- **設定の永続化** - Cookieに保存（1年間有効）

## ⚙️ 設定方法

「設定」リンクからタイマー設定画面を開き、GUIでタイマーを追加・編集できます。
上級者向けにJSON直接編集も可能です。

### 設定例

```json
[
  {"title": "起床時刻", "targetTime": "08:20:00"},
  {"title": "勤務開始時刻", "targetTime": "09:00:00"},
  {"title": "定時まで", "targetTime": "18:00:00"},
  {"title": "就寝時間まで", "targetTime": "02:20:00"},
  {"title": "2025年残り", "targetTime": "2026-01-01T00:00:00"}
]
```

### 形式

| 形式 | 例 | 説明 |
|------|-----|------|
| 時刻のみ | `"08:20:00"` | 毎日その時刻にリセット |
| 日時 | `"2024-01-01T00:00:00"` | 特定日時まで（日数も表示） |

## 📝 注意事項

- 設定はブラウザのCookieに保存されます
- Cookieを削除すると設定がリセットされます
- 1年以上ページを開かないと設定の有効期限が切れます
- タイトルに半角スペースを入れると動作が不安定になる場合があります

## 💰 コストタイマー - ミーティングコスト可視化

https://velaciela-ringWing2007.github.io/cost-timer/

ミーティングの参加者と時給からコストをリアルタイムで計算・表示します。

### ✨ 機能

- **リアルタイムコスト計算** - 参加者の時給から現在のコストを自動計算
- **時間管理**
  - 経過時間・残り時間・超過時間を表示
  - プログレスバー（超過時は赤色に変化）
  - 予定コストと実際のコストの比較
- **参加者プリセット** - よく使う参加者構成を保存・呼び出し
- **タイマー画面から直接編集** - 参加者の追加・削除・変更が可能
- **10種類のテーマ切り替え** - 気分や場面に合わせてUIを変更

### 🎨 テーマ一覧

| テーマ | 特徴 |
|--------|------|
| Default | ダークブルー（標準） |
| Terminal Green | マトリックス風、スキャンライン効果 |
| NERV Orange | 警告システム風、六角形UI |
| Neon Cyber | ピンク×シアン、ネオングロー |
| Midnight Purple | 深夜のパープル |
| Ocean Deep | 深海ブルー |
| Sunset | サンセットオレンジ |
| Retro Amber | レトロCRT風 |
| Pop Pink | パステルピンク、かわいい系 |
| Pop Aqua | パステル水色、やさしい系 |

### ⚙️ 設定

設定画面で参加者プリセットを管理できます。年収から時給を計算する参考表も掲載。

## 🕐 時計プリント - 時間学習用

https://velaciela-ringWing2007.github.io/clockPrintPage.html

子供に時間を教えるための、印刷可能なアナログ時計です。

### ✨ 機能

- **アナログ時計表示** - SVGで高品質な時計を描画
- **時刻設定** - 時・分・秒を自由に設定可能
- **針の色カスタマイズ** - カラーピッカーで短針・長針・秒針の色を変更
- **表示オプション**
  - 数字の表示/非表示
  - 分の目盛りの表示/非表示
  - 秒針の表示/非表示
  - デジタル時刻の表示/非表示
- **A4印刷対応** - 印刷ボタンでA4サイズに最適化して印刷

### 🎨 デフォルト設定

| 針 | 色 |
|----|-----|
| 短針（時） | 🔴 赤 |
| 長針（分） | 🔵 青 |
| 秒針 | 🟢 緑 |
